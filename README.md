## 🛠️ Run Locally

Clone the project

```bash
  git clone https://github.com/SaiNivedh26/Team-Nooglers
```

Go to the project directory

```bash
  cd Team-Nooglers
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  npm run start
```
